# PRODIGY_WD_01
This project is a modern, responsive landing page featuring an interactive navigation menu. The goal was to build a seamless user experience using core web technologies, focusing on fixed positioning and dynamic styling based on user interaction.

🌟 Key Features
Fixed Navigation: The menu stays at the top of the viewport, ensuring easy access regardless of scroll depth.

Scroll-Triggered Interactivity: Using JavaScript, the navigation bar changes its background color and styling once the user scrolls down, improving readability against page content.

Hover Effects: CSS-driven transitions that provide instant visual feedback when users interact with menu items.

Fully Responsive: Designed to look and function perfectly across desktops, tablets, and mobile devices.

🛠️ Technologies Used
HTML: Semantic structure for the navigation and page sections.

CSS: Advanced styling, including position: fixed and smooth transition effects.

JavaScript: Event listeners for window scrolling to toggle active classes dynamically.
